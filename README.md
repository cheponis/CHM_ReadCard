This is the code used to read all 80-column IBM punched cards that came with the Purdue Manotes (sp?) Collection for the IBM 1620.  Four full dual-tray cabinets were full of cards; and after reading them all in, they fit on a single 1.44 MB Floppy Diskette when packed into a .ZIP file.  (!)

The reader program used a DOS computer, Microsoft DOS C++ compiler, and an external RS-232 attached Card Reader operating at 19,200 bps.
